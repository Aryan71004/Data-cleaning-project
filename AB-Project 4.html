Question 9:

Use .plot() to create a bar chart to show average profit in each 'Country' for 'Accessories' Product_Category.

  • Sort it in ascending order.
  • Use seaborn's 'Set2' palette for the color.
  • Use matplotlib (plt.) to add title:'Average profit in each Country for Accessories Product_Category', x label:'Country', and y label :'Average profit'.
  • Add labels for each bar. The labels should show the average profit with 2 decimal places and should have $ sign.
In[42]:
accessories_df = df1[df1['Product_Category'] == 'Accessories']
avg_profit_per_country = accessories_df.groupby('Country')['Profit'].mean().sort_values()
sns.set_palette('Set2')
plt.figure(figsize=(10, 6))
ax = avg_profit_per_country.plot(kind='bar')
# Add title and labels
plt.title('Average Profit in Each Country for Accessories Product_Category')
plt.xlabel('Country')
plt.ylabel('Average Profit')
# Adding labels to each bar (showing average profit with 2 decimal places and a $ sign)
for i, v in enumerate(avg_profit_per_country):
    ax.text(i, v + 5, f'${v:.2f}', ha='center', va='bottom')
# Show the plot
plt.tight_layout()
plt.show()

explode_list = [0.1, 0, 0, 0, 0, 0] # ratio for each street with which to offset each wedge. Since we have 8 streets, I put 8 values in the list. 
#plt.title('Percentage of orders in each Country for Accessories Product_Category')

# Filter data for 'Accessories' Product_Category
accessories_data = df1[df1['Product_Category'] == 'Accessories']

# Group by 'Country' to count the orders
df1 = accessories_data['Country'].value_counts()

# Plot the pie chart using the requested format
plt.figure(figsize=(8, 8))

df1.plot(kind='pie',
         autopct='%0.1f%%', # add percentages with 1 decimal place
         startangle=90,     # start angle at 90 degrees
         shadow=True,       # add shadow
         explode=explode_list # explode the 'United States' slice
         )

# Add a legend and ensure layout does not overlap
plt.legend(title="Country", loc="upper right", bbox_to_anchor=(1.2, 1))
plt.tight_layout()

# Show the plot
plt.show()
